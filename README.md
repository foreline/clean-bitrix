# Bitrix implementation of Clean Framework

https://github.com/foreline/clean

## Installation

composer.json:
```json
{
  "require": {
    "php": ">=8.0",
    "foreline/clean-bitrix": "dev-main"
  },
  "repositories": [
    {
      "type": "git",
      "url": "https://github.com/foreline/clean-bitrix.git"
    }
  ]
}
```
